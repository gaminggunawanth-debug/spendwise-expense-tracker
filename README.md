# 💸 SpendWise - Personal Expense Tracker

A beautiful and simple expense tracking web app built with Python and Streamlit.

![SpendWise Preview](Screenshot 2026-04-09 163813.png)

## ✨ Features

- Add daily expenses with date and category
- Beautiful dark modern UI
- Interactive Pie Chart (Spending by Category)
- Daily spending trend line chart
- Total spent, average, and transaction count
- Fully responsive and easy to use

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Plotly (for interactive charts)
